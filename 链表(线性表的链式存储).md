#### 链表结构体定义

```c
typedef struct lnode {
    int data;
    struct lnode *next;
}lnode,*linklist;
//
linklist p;
lnode *l;
//这两句话效果一样
lnode *l;
l=(lnode*)malloc(sizeof(lnode));
```
