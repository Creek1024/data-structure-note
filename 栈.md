### Stack（栈）

#### 只允许一端插入或删除的线性表称为栈

#### 栈顶top：插入删除端

#### 栈底bottom：固定，不允许插入删除的另一端；



#### 栈的数据体定义

```c
#define Maxsize 50
typedef struct Stack{
    int data[Maxsize];
    int top;
}Stack;
```

#### 初始化栈（初始化栈顶指针）

```c
void initStack(Stack &s){

s.top=-1;

}
```
